## 入门

### 首次启动

首次启动应用时，您会看到一个空的音乐库。按照以下步骤添加您的音乐：

1. 点击侧边栏中的 **"添加文件夹"** 按钮
2. 在弹出的对话框中选择包含音乐文件的文件夹
3. 应用将自动扫描文件夹中的所有音频文件
4. 扫描完成后，您的音乐将显示在库中

### 界面概览

应用界面分为四个主要区域：


## 音乐库管理

### 添加文件夹

您可以通过以下方式添加音乐文件夹：

- 点击侧边栏的 **"添加文件夹"** 按钮
- 使用快捷键 `Ctrl+O` (Windows/Linux) 或 `Cmd+O` (macOS)

应用支持以下音频格式：

- MP3, FLAC, WAV, AAC, OGG
- M4A, WMA, AIFF, ALAC, OPUS

### 刷新文件夹

如果您在文件夹中添加或删除了音乐文件，可以刷新文件夹以更新库：

1. 在音乐库视图中找到要刷新的文件夹
2. 点击文件夹旁边的刷新图标
3. 等待重新扫描完成

### 删除文件夹

从库中移除文件夹：

1. 在音乐库视图中找到要删除的文件夹
2. 点击文件夹旁边的删除图标
3. 确认删除

> **注意**: 删除文件夹只会从库中移除引用，不会删除实际的音乐文件。

### 查看方式

您可以通过不同方式浏览音乐：

- **音乐库** - 按文件夹查看所有歌曲
- **专辑** - 按专辑分组查看
- **艺术家** - 按艺术家分组查看

## 播放控制

### 基本操作

| 操作      | 方法                           |
| --------- | ------------------------------ |
| 播放歌曲  | 双击歌曲，或选中后按空格键     |
| 暂停/继续 | 点击播放/暂停按钮，或按空格键  |
| 上一曲    | 点击上一曲按钮                 |
| 下一曲    | 点击下一曲按钮                 |
| 调整进度  | 拖动进度条，或使用左右箭头键   |
| 调整音量  | 拖动音量滑块，或使用上下箭头键 |

### 播放模式

点击播放模式按钮可以切换不同的播放模式：

| 模式     | 图标 | 说明                   |
| -------- | ---- | ---------------------- |
| 顺序播放 | ➡️   | 按顺序播放队列中的歌曲 |
| 随机播放 | 🔀   | 随机选择下一首歌曲     |
| 单曲循环 | 🔂   | 重复播放当前歌曲       |
| 列表循环 | 🔁   | 播放完队列后从头开始   |

### 进度控制

- **点击进度条**: 跳转到指定位置
- **拖动进度条**: 精确控制播放位置
- **左箭头键**: 后退 5 秒
- **右箭头键**: 前进 5 秒

### 音量控制

- **拖动音量滑块**: 调整音量
- **点击静音按钮**: 快速静音/取消静音
- **上箭头键**: 增加音量
- **下箭头键**: 降低音量

## 播放列表

### 创建播放列表

1. 点击侧边栏的 **"新建播放列表"** 按钮，或使用 `Ctrl+N` / `Cmd+N`
2. 输入播放列表名称
3. 点击确认创建

### 添加歌曲到播放列表

方法一：右键菜单

1. 右键点击歌曲
2. 选择 **"添加到播放列表"**
3. 选择目标播放列表

方法二：拖拽

1. 选中一首或多首歌曲
2. 拖拽到侧边栏的播放列表上

### 管理播放列表

在播放列表详情页面，您可以：

- **重新排序**: 拖拽歌曲调整顺序
- **删除歌曲**: 点击歌曲旁边的删除按钮
- **重命名**: 点击播放列表名称进行编辑
- **删除播放列表**: 点击删除按钮

> **注意**: 从播放列表删除歌曲不会删除原始音乐文件。

## 播放队列

### 查看播放队列

点击播放控制栏右侧的队列图标，打开播放队列面板。

### 管理队列

| 操作           | 方法                       |
| -------------- | -------------------------- |
| 添加到队列末尾 | 右键歌曲 → "添加到队列"    |
| 下一首播放     | 右键歌曲 → "下一首播放"    |
| 从队列移除     | 在队列面板中点击删除按钮   |
| 调整顺序       | 在队列面板中拖拽歌曲       |
| 清空队列       | 点击队列面板的 "清空" 按钮 |

### 队列与播放列表的区别

- **播放队列**: 临时的播放列表，关闭应用后会保存当前状态
- **播放列表**: 永久保存的歌曲集合，可以随时播放

## 搜索与筛选

### 搜索

1. 点击搜索框，或使用 `Ctrl+F` / `Cmd+F`
2. 输入搜索关键词
3. 搜索结果会实时显示

搜索范围包括：

- 歌曲标题
- 艺术家名称
- 专辑名称

### 筛选

在专辑或艺术家视图中：

- 点击专辑查看该专辑的所有歌曲
- 点击艺术家查看该艺术家的所有歌曲和专辑

## 歌词显示

### 支持的歌词格式

应用支持 LRC 格式的歌词文件，包含时间标签的歌词会自动同步滚动。

### 歌词来源

应用会按以下顺序查找歌词：

1. **内嵌歌词** - 首先尝试从音频文件元数据中提取歌词
2. **外部 LRC 文件** - 如果没有内嵌歌词，会查找与音频文件同名的 `.lrc` 文件

例如，播放 `song.mp3` 时，会查找 `song.lrc`。

### 歌词面板

点击播放控制栏的歌词按钮可以打开/关闭歌词面板：

- 歌词会随播放进度自动滚动
- 当前播放的歌词行会高亮显示
- 如果没有找到歌词，会显示"暂无歌词"

### LRC 文件格式示例

```
[00:00.00]歌曲名称
[00:05.50]第一句歌词
[00:10.20]第二句歌词
[00:15.80]第三句歌词
```

## 设置与个性化

### 主题切换

应用支持亮色和暗色主题：

1. 点击侧边栏底部的主题切换按钮
2. 选择 **亮色**、**暗色** 或 **跟随系统**

### 数据存储

应用会自动保存以下数据：

- 音乐库配置（已添加的文件夹）
- 播放列表
- 播放状态（当前歌曲、播放位置、音量）
- 播放模式偏好
- 主题设置

数据存储在本地，不会上传到任何服务器。

## 常见问题

### Q: 为什么有些歌曲没有显示封面？

A: 如果音频文件没有嵌入封面图片，应用会显示默认占位图。您可以使用音频标签编辑软件为音乐文件添加封面。

### Q: 如何更新已添加文件夹中的新歌曲？

A: 点击文件夹旁边的刷新按钮，应用会重新扫描文件夹并更新库。

### Q: 删除播放列表会删除音乐文件吗？

A: 不会。删除播放列表只会删除播放列表本身，原始音乐文件不受影响。

### Q: 应用支持哪些音频格式？

A: 支持 MP3, FLAC, WAV, AAC, OGG, M4A, WMA, AIFF, ALAC, OPUS 等常见格式。

### Q: 如何备份我的播放列表？

A: 播放列表数据存储在应用数据目录中。您可以备份该目录来保存所有设置和播放列表。

数据目录位置：

- Windows: `%APPDATA%/orange-music`
- macOS: `~/Library/Application Support/orange-music`
- Linux: `~/.config/orange-music`

### Q: 应用启动很慢怎么办？

A: 如果您的音乐库很大，首次加载可能需要一些时间。应用会缓存数据，后续启动会更快。
