# 更新日志

本文档记录 OrangeMusic 的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-01-15

### 🎉 首次发布

OrangeMusic 的首个正式版本，提供完整的本地音乐播放功能。

### ✨ 新增功能

#### 音乐库管理

- 支持添加本地文件夹到音乐库
- 自动扫描文件夹中的音频文件
- 支持 MP3, FLAC, WAV, AAC, OGG, M4A, WMA, AIFF, ALAC, OPUS 格式
- 自动解析音频文件元数据（标题、艺术家、专辑、封面等）
- 支持刷新和删除已添加的文件夹
- 显示扫描进度

#### 音频播放

- 完整的播放控制（播放、暂停、停止）
- 上一曲/下一曲切换
- 进度条拖动和点击跳转
- 音量控制和静音
- 多种播放模式：顺序播放、随机播放、单曲循环、列表循环

#### 歌词显示

- 支持 LRC 格式歌词文件
- 自动从音频文件元数据提取内嵌歌词
- 支持外部 .lrc 歌词文件（与音频文件同名）
- 歌词自动同步滚动
- 当前播放行高亮显示

#### 播放列表

- 创建、重命名、删除播放列表
- 添加歌曲到播放列表
- 从播放列表移除歌曲
- 拖拽调整歌曲顺序
- 播放列表自动持久化

#### 播放队列

- 查看当前播放队列
- 添加歌曲到队列末尾
- "下一首播放"功能
- 从队列移除歌曲
- 拖拽调整队列顺序
- 清空队列

#### 搜索与筛选

- 实时搜索歌曲、专辑、艺术家
- 按专辑浏览
- 按艺术家浏览
- 搜索结果高亮

#### 用户界面

- 现代化的界面设计
- 响应式布局
- 亮色/暗色主题切换
- 跟随系统主题
- 虚拟化长列表，流畅处理大型音乐库
- 专辑封面显示
- 默认封面占位图

#### 键盘快捷键

- `Space` - 播放/暂停
- `←` / `→` - 后退/前进 5 秒
- `↑` / `↓` - 增加/降低音量
- `Ctrl/Cmd + F` - 聚焦搜索框
- `Ctrl/Cmd + N` - 新建播放列表
- `Ctrl/Cmd + O` - 添加文件夹

#### 数据持久化

- 自动保存音乐库配置
- 自动保存播放列表
- 自动保存播放状态（当前歌曲、位置、音量）
- 自动保存用户设置（主题、播放模式）
- 应用重启后恢复上次状态

#### 错误处理

- 播放错误自动跳过并播放下一首
- 文件不存在提示
- 元数据解析失败降级处理
- 全局错误边界
- Toast 通知

### 🛠️ 技术栈

- Electron + Vite + React 19 + TypeScript
- Tailwind CSS 4 + shadcn/ui
- Zustand 状态管理
- Howler.js 音频引擎
- music-metadata 元数据解析
- react-virtuoso 虚拟化列表
- electron-store 数据持久化
- Vitest + fast-check 测试框架

### 📦 支持平台

- Windows 10+
- macOS 10.15+
- Linux (Ubuntu 18.04+)
